Raw Authentication Callback

This will walk through the authorization process, using no classes from the ORCID client except for the context. Hence, "raw".

Click the link, and you will be sent to the ORCID page, which will ask you to log in, and authorize the authentication. Then you will be sent back here, where you will see the code that was issued.

A form is prepared to trade that code for an access token. You can inspect the contents of the form, or modify them, before sending it.

Finally, if everything goes well, you will see the raw XML that contains the access token.

This is what we will send:

Request URL:

{{authRequestUrl}}

Do raw authentication.
